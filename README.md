# java-rpg

## What is this project?

A bit of backstory: I am a student at the University of Applied Sciences and Art Ghent. At the time of making this project, I am studying for my professional bachelor's degree in Computer Sciences.

I have a course on java *(ewww)* programming, so as a way to challenge myself, and apply what I've learned during class, I wanted to make my own project.

I love old RPG titles on the NES, SNES, Gameboy.. And was inspired when playing through Final Fantasy I - Pixel Remaster. So I decided to make an RPG in java.<br>
The goal is to use no fancy libraries, except for JavaFX.<br>Everything must be coded in Java, and I use XML to load in enemies / attacks..

Follow along if you are interested in making a RPG-game, without any fancy game engines.

In the future, ofcourse UI & graphics will be added, although they might be bare-bones. *Since I am quite back-end minded*<br>
However, my reasoning is simple: "If I can't make it work in the CLI, then I dont see the use of adding fancy buttons, that would execute the same functions.

Thank you for being interested / following along!
